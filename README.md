# 🛒 Zepto Sales Analysis Dashboard

## 📌 Overview  
This project presents an interactive **Power BI dashboard** analyzing Zepto sales data to uncover insights on revenue, product performance, and customer behavior.

## 📊 Key Insights  
- Tracked key KPIs: Total Sales, Average Sales, and Item Count  
- Analyzed category-wise and outlet-wise performance  
- Identified customer rating trends and product demand patterns  
- Visualized sales trends for better decision-making  

## 🛠️ Tools & Technologies  
- Power BI  
- DAX (Data Analysis Expressions)  
- Power Query  

## ⚙️ Data Processing  
- Cleaned and transformed raw data using Power Query  
- Created calculated measures using DAX  
- Designed interactive dashboard with filters and slicers
